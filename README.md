
# Лемана Про — простая юнит-экономика

## Что делает
Пользователь загружает:
1. каталог товаров,
2. файл комиссий,
3. файл логистики,

и получает:
- автоматически определенную категорию/шаблон,
- комиссию,
- логистику по габаритам,
- одну среднюю логистику по долям зон,
- одну рекомендованную цену продажи.

## Как запускать
```bash
pip install streamlit pandas openpyxl openai
streamlit run app.py
```

## Какой нужен каталог
Нужны колонки:
- Артикул / SKU
- Наименование / Название
- Длина
- Ширина
- Высота
- Вес
- Себестоимость

Названия могут немного отличаться: приложение пытается подобрать их автоматически.

## Какие нужны файлы Лемана Про
### Файл комиссий
Лист вроде:
- `Комиссия_FBS и FBO`
или похожий.

### Файл логистики
Листы вроде:
- `Тарифы (Последняя миля)`
- `Тарифы (Доставка до СЦ)` или `Тарифы (услуга Нулевая миля)`

## Как обновлять тарифы
Ничего в коде менять не нужно.
Просто каждый раз загружайте новый Excel от Лемана Про в интерфейсе приложения.

## Важно
- Склад Москва для FBS = `Зона откуда 9`
- Одна цена считается по средней структуре зон:
  - Москва/МО
  - СПБ/ЛО
  - Регионы
