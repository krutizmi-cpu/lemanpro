Что изменено в этой версии
- Выгрузка результата и шаблона теперь в Excel (.xlsx), а не CSV.
- Добавлены рекомендованные цены по зонам для FBS:
  - Москва и МО
  - СПБ и ЛО
  - Регионы (среднее по всем региональным зонам в тарифном файле)
  - Средневзвешенная цена / одна цена к публикации
- Добавлены средние KPI по рекомендованным ценам сверху.
- Для одной цены к публикации используется средневзвешенная модель зон.
- Округление рекомендованной цены настраивается в sidebar.

Как использовать
1. Замените старый app.py на новый app.py из архива.
2. Запустите:
   streamlit run app.py
3. В sidebar для FBS задайте доли зон, например 70 / 20 / 10.
4. После расчета скачайте Excel-выгрузку.

Как считается одна цена
- Москва и МО: отдельная рекомендованная цена
- СПБ и ЛО: отдельная рекомендованная цена
- Регионы: средняя рекомендованная цена по всем доступным региональным зонам
- Одна цена к публикации: средневзвешенная цена по долям зон из sidebar
